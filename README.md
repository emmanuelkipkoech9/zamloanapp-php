# 💰 ZamLoans - Quick Zambian Loans Platform

A modern, mobile-friendly loan application platform for Zambian users to apply for loans, pay withdrawal fees via M-Pesa, and get funded quickly.

##  Features

- **Loan Selection** - Choose from 16 loan amounts ranging from ZMW 500 to ZMW 20,000
- **Application Form** - Submit personal details with validation
- **M-Pesa Integration** - Pay withdrawal fees via STK push using NestLink API
- **Real-time Status** - Track payment and application status
- **Responsive Design** - Works seamlessly on mobile and desktop
- **Analytics Tracking** - Track user actions and payment attempts
- 

##  Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript
- **Payments:**  API (M-Pesa integration)
- **Deployment:** Render, Docker, PHP
- **Fonts:** Google Fonts (DM Sans)
